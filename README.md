## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AdkxKashyap/Socialize-Client.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `src/environments/environvent.ts`.
    Note - Do not change Port
   ```js
    googleClientID = YOUR GOOGLE CLIENT ID. PLEASE CHECK GOOGLE DEV CONSOLE
   ```
5. Start Server
    ```sh
   ng s
   ```
